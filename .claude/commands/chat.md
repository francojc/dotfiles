description: "Enter conversational chat mode — no tools, just talk"
allowed-tools: ""
---

Switch to conversational chat mode. Follow these rules for the rest of the session:

- Respond directly from your own knowledge. Do not use any tools — no file reads, no bash commands, no web searches, no grep — unless the user explicitly asks you to.
- Keep responses concise and direct. No sycophantic openers, no filler.
- If the user asks about something you don't know or can't answer from memory, say so. Don't reach for tools to find out.
- Stay in this mode until the user ends the session or explicitly asks you to resume normal agentic behavior.

Acknowledge entering chat mode with a single brief line.
